Welcome to League Spectate

League Spectate is a lightweight tool/Graphical frontend written in C#. Essentially it enables anyone with a League of Legends client and an acccount in the proper region to get the spectator/observer data for any public game and spectate it right from the League of Legends client. As an extra function you can load "*.bat" files from op.gg/spectate/ and fow.kr (its like a korean lolking). Those crash most of the EU clients in my experience by default but you can load them without any problems with League Spectate.

Usage

  1. Download .NET 4.5 if you haven't installed it before. http://www.microsoft.com/en-GB/download/details.aspx?id=30653

  2. Download League Spectate (Zip Folder on the top of this page)

  3. The main application is Bin/Spectate.exe

--- Spectating op.gg / fow.kr --- 1. Go here http://www.op.gg/spectate/pro/ 2. Choose a game and press the green "Spectate" button 3. It will download a bat file 4. Load up League Spectate, open the "Extras" window 5. Browse from op.gg bat and browse the bat file you just downloaded

  1. Go here http://fow.kr/
  2. Search for a summoner (You can take "SKT T1 Faker" for example)
  3. Choose a game and press the green "Play" button
  4. It will download a bat file
  5. Load up League Spectate, open the "Extras" window
  6. Browse from fow.kr bat and browse the bat file you just downloaded
